# 證照題庫練習系統 - 快速開始指南

## 🚀 立即開始

### 1️⃣ 安裝依賴
在終端機執行：
```bash
flutter pub get
```

### 2️⃣ 啟動應用程式
```bash
flutter run
```

### 3️⃣ 匯入題目
有兩種方式：

**方式 A：使用內建範例**
1. 點擊「匯入題庫」
2. 點擊「匯入範例題目」
3. 自動匯入 5 道 Flutter/Dart 相關題目

**方式 B：匯入自訂題庫**
1. 準備 JSON 格式題庫（參考 sample_questions.json）
2. 點擊「匯入題庫」→「從檔案匯入」
3. 選擇您的 JSON 檔案

### 4️⃣ 開始練習
1. 返回首頁
2. 點擊「開始練習」
3. 選擇答案並提交
4. 查看解析和統計

### 5️⃣ 複習錯題
1. 首頁點擊「錯題複習」
2. 查看錯題列表
3. 重新練習錯題

## 📁 JSON 題庫範例

```json
[
  {
    "questionText": "您的題目內容？",
    "options": [
      "選項 A",
      "選項 B",
      "選項 C",
      "選項 D"
    ],
    "correctAnswer": 0,
    "explanation": "答案解析說明（可選）",
    "category": "類別名稱"
  }
]
```

**重要提醒：**
- `correctAnswer` 是索引值（0, 1, 2, 3）
- `options` 至少要有 2 個選項
- `explanation` 可以省略

## 💡 小技巧

1. **清除練習記錄**：首頁有「清除練習記錄」按鈕，可重置統計
2. **重新整理統計**：點擊統計頁面右上角的重新整理圖示
3. **練習模式**：題目會隨機打亂，增加練習變化
4. **錯題複習**：專注於答錯的題目，提高學習效率

## 🛠 故障排除

**Q: 匯入題庫後看不到題目？**
A: 檢查 JSON 格式是否正確，每個欄位都必須填寫

**Q: 無法開始練習？**
A: 確認已經匯入至少一道題目

**Q: 錯題複習按鈕無法點擊？**
A: 表示目前沒有答錯的題目，先進行一般練習
