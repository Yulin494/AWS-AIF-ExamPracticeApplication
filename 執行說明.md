# 執行應用程式說明

## 問題說明

您遇到空白畫面的原因：

1. **Windows 桌面版**：需要啟用開發者模式
2. **Web 版本**：sqflite 資料庫不支援 Web 平台

## 解決方案

### 方案 1：啟用 Windows 開發者模式（推薦用於桌面應用）

1. 開啟 Windows 設定
2. 前往「更新與安全性」→「開發人員選項」
3. 啟用「開發人員模式」
4. 執行：`flutter run -d windows`

### 方案 2：使用 Android 模擬器或實體裝置

1. 開啟 Android Studio 的 AVD Manager
2. 啟動一個 Android 模擬器
3. 執行：`flutter run`

### 方案 3：使用 iOS 模擬器（Mac 系統）

1. 開啟 Xcode
2. 啟動模擬器
3. 執行：`flutter run -d ios`

## 目前狀況

Web 版本目前無法正常運作，因為：
- sqflite 資料庫套件不支援 Web 平台
- 需要使用其他存儲方案（如 IndexedDB、shared_preferences 等）

## 建議

**最快的測試方式：**
1. 啟用 Windows 開發者模式（一次性設定）
2. 重新執行：`flutter run -d windows`

或者

1. 使用 Android 模擬器
2. 執行：`flutter run`

## 需要修改以支援 Web？

如果您希望在 Web 上運行此應用，我需要：
1. 將 sqflite 替換為 shared_preferences 或 IndexedDB
2. 修改資料存儲邏輯
3. 調整檔案匯入功能

請告訴我您希望：
- A. 在 Windows 桌面上執行（啟用開發者模式後即可）
- B. 在 Android/iOS 模擬器上執行
- C. 修改程式碼以支援 Web 版本
